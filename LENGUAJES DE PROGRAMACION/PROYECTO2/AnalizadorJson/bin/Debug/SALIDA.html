The leading open source automation server, Jenkins provides hundreds of plugins to support building, deploying and automating any project. As an extensible automation server, Jenkins can be used as a simple CI server or turned into the continuous delivery hub for any project... Jenkins is a self contained Java based program, ready to run out of the box, with packages for Windows, Mac OS X and other Unix like operating systems... Jenkins can be easily set up and configured via its web interface, which includes on the fly error checks and built-in help.
